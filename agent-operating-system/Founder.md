## Distinctive technical signature

What makes Joe unusual relative to either pure engineers or pure software people:

- **Production bridge engineer** with FDOT/AASHTO experience (Kisinger Campo era); has actually drawn the calcs, specced the plans, walked the inspections.
- **Forensic investigator** with insurance/legal report-writing chops (NV5 era); knows what regulators, carriers, and lawyers expect from a structural narrative.
- **Geotechnical field operator**; has done the soil sampling and SPT work himself, not just specified it.
- **Published structural researcher**; peer-reviewed coauthor on epoxy + carbon-fiber repair of corroded prestressed concrete (see Publications below).
- **Working software engineer**; Python (FastAPI, Flask, asyncio), live REST + WebSocket APIs, deployed Linux services on cloud infrastructure, integrated Anthropic and OpenAI APIs into autonomous decision-making systems.
- **Field-equipment operator**; FAA Part 107 commercial UAS for inspection; AAUS scientific research diver for underwater structural inspection.

This combination, a civil engineer who can build the software AND operate the inspection equipment, is what Rarefied Earth's positioning is built on.
